# Shared assets

CC0 / royalty-free assets you reuse across games.

## Recommended sources

- https://kenney.nl/assets — CC0, huge variety, well-organized
- https://opengameart.org — mixed licenses, check each
- https://freesound.org — sound effects (check license)
- https://fonts.google.com — fonts (Open Font License)

## Conventions

- One folder per pack: `shared/assets/kenney-ui-pack-1/`
- Keep LICENSE.txt with each pack
- Don't modify packs in-place — copy to a game's local assets if you tweak
