# Improve a Prompt

## When to use

After a prompt produces mediocre output, before deleting or rewriting.

## The prompt

Here is a prompt I use:

```
{{PROMPT_TEXT}}
```

Here's the output it produced (which was unsatisfying):

```
{{OUTPUT}}
```

Here's what I actually wanted:

{{WHAT_I_WANTED}}

Suggest specific edits to the prompt to make it more likely to produce
what I wanted. Focus on:
- Missing constraints
- Ambiguous instructions
- Missing examples
- Wrong output format

Give me the rewritten prompt, not just suggestions.
