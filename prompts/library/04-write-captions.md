# Write Captions (TikTok / Shorts)

## When to use

After cutting clips, when you need captions for TikTok, YouTube Shorts, and
itch.io devlog posts.

## Input variables

- {{GAME_NAME}}: e.g., "Coffee Shop Idle"
- {{CONCEPT_PITCH}}: the one-line pitch
- {{CLIP_DESCRIPTION}}: what happens in the clip (e.g., "first auto-brewer
  unlock — number starts climbing on its own")
- {{DURATION}}: clip length in seconds
- {{PLATFORM}}: tiktok / shorts / itch

## The prompt

I'm an indie dev shipping small browser games and posting dev content.

Game: {{GAME_NAME}}
Pitch: {{CONCEPT_PITCH}}
Clip ({{DURATION}}s): {{CLIP_DESCRIPTION}}
Target platform: {{PLATFORM}}

Write three caption variants. Each variant should:

1. Open with a hook in the first 4 words (this is what shows in feed previews)
2. Be under 150 characters for TikTok/Shorts, under 280 for itch
3. End with a soft CTA (not "follow me" — something like "playable link in bio"
   or "trying to ship 10 of these this year")
4. Include 3-5 relevant hashtags at the end (no #fyp spam)

Variants should differ in tone:
- Variant A: dry / matter-of-fact (devlog style)
- Variant B: hype / energetic (algorithm bait, but not cringe)
- Variant C: question / curiosity (hook viewer to comment)

After the variants, give me your pick and one sentence why.
